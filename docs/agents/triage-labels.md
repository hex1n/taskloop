# Triage Labels

The engineering skills use these canonical roles as local `Status:` values.

| Canonical role | Local status | Meaning |
|---|---|---|
| `needs-triage` | `needs-triage` | Maintainer evaluation required |
| `needs-info` | `needs-info` | Waiting for more information |
| `ready-for-agent` | `ready-for-agent` | Fully specified and agent-grabbable |
| `ready-for-human` | `ready-for-human` | Requires human action |
| `wontfix` | `wontfix` | Will not be actioned |
